# InstallTi84EmulatorLinux
Install ti-84 emulator on linux

```sudo apt-get install libgtk-2.0-dev```

```sudo apt-get install libticalcs-dev```

```sudo apt-get install libticonv-dev```

```cd tilem/gui``` 

makefile --> ```LIBS = -lm```

``` ./configure && make && sudo make install```
